# Fahmi Harun — Personal Website

Static personal website for Fahmi Harun, focused on software development, backend, mobile, Linux infrastructure, and AI-assisted engineering.

## Files

- `index.html` — deployable website entry point for GitHub Pages.
- `Personal Website.html` — original local website file copy.
- `Fahmi_Harun_CV_Astra_Bahasa_Indonesia.pdf` — downloadable CV linked from the website.

## Preview

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
